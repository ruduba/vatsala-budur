I am very interested in the study of the impact of Artificial Intelligence on academics and human cognition. This website is a platform for me to express my thoughts and findings on the same.

Why did I make this website so… um… look so outdated? I have been on the internet since I was 3 (that's a really long time ago), and the internet at that time had so much personality. Every website you visited felt like visiting a completely new corner of the internet. But now, every website is the same, the same old gradients, the same old overdone template, very corporate, too professional for my taste. I didn’t want my website to look like that, I wanted to make something that reflected who I am and expressed my interests.

So behold, welcome to my website (I have not purchased a domain name yet, probably hosting this on github pages lol). This place is reminiscent of the old days, the simpler times. To be honest, this feels like home. Go ahead, explore!

Projects

  • Console FPS

    A windows console based First Person Shooter, runs on the ScreenBuffer. Has WASD controls (Forward, Rotate Left, Backward, Rotate Right), and Strafing controls (Q, E). Includes Collision Detection (Movement Negation) and also has Depth Perspective with Ray Casting. Everything you see on screen is an ASCII Character or WhiteSpace. A map is provided on the top left for easier navigation.
    Built with: C++

  • VatChain

    An interative CLI based Blockchain built using Java and SHA-256 hashing
    You can do the following
    • Check balances
    • Send vatCoins
    • Request vatCoins
    • Mine block
    • Show blockchain
    • Check validity
    • Exit

  • Souper Scraper

    A Price Scraper and Comparison Dashboard. Scrapes prices from two competitive websites and compares price Rupees/Gram. Also highlights missing items in either site available on the other.
    Built using: Python, Beautiful Soup, WebDriver, Selenium

  • ASCII Web Cam

    Click to View Demo

    This takes your webcam feed and turns it into an ASCII video output. It is drawn directly on the DOM, you can flip orientation of the output, capture a particular frame (like a camera), and also be able to copy the frame.
    Built with: p5.js, HTML, CSS

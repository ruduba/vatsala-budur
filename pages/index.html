The Better Internet Project

My main goal for the rest of 2026, is to create a better internet, more specifically, a human internet. This will be through The Better Internet Project. I believe that if a human being did not care to take the time and effort to create something by themselves, there is no use in consuming it, be it written text, generated images, art, videos, or even code. You can read more about this in the Blog or check out The Better Internet Project.
